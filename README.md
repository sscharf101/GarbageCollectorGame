A simple and fun Python game where you collect recyclables and avoid trash!

## How to Play

- **Goal**: Collect as many green recyclable items as possible while avoiding red trash
- **Controls**:
  - Arrow keys (‚Üê ‚Üí) or A/D keys to move left and right
- **Scoring**:
  - Catch green recyclables: +10 points
  - Catch red trash: -5 points
  - Miss 10 recyclables: Game Over!

## Installation

1. Install the required dependency:
```bash
pip install -r requirements.txt
```

2. Run the game:
```bash
python garbage_game.py
```

## Game Features

- Colorful graphics with recyclable and trash items
- Progressive difficulty (items fall faster as you score more)
- Score tracking
- Game over when you miss too many recyclables
- Restart option
